## Objective

Design and configure a secure cybersecurity home lab environment for practicing web application security testing and network analysis in an isolated setup.

## Environment Used

- Kali Linux
- VirtualBox
- Docker
- OWASP Juice Shop

## Security Tools Used

- Nmap
- Burp Suite
- Wireshark

- Configured Kali Linux virtual machine
- Deployed OWASP Juice Shop using Docker
- Verified network communication
- Performed service discovery using Nmap
- Intercepted web requests using Burp Suite
- Captured and analyzed traffic using Wireshark
